<p>An essential tool for Android Developers which instantly displays the package name and class name of the application currently in the foreground.</p><p>HOW DOES IT WORK</p><p>We use package usage stats to monitor app activity changes and show the info in a freely moveable popup window. In the global version which is available in GitHub, we also use AccessibilityService to further increase the performance of monitoring.</p><p>APP FEATURES</p><ul><li>Provides a freely moveable popup window to view current activity info.</li><li>Provides a notification to view current activity info in pages where popup window can't be shown.</li><li>Keeps track of activities you visited before in history.</li><li>Supports copying text from popup window and notification.</li><li>Supports quick settings for easy access to the popup window from any place in your device.</li></ul><p>KEEP CALM & PRIVACY</p><p>Current Activity doesn’t require root or any specific requirements. It respects system security and user privacy. Any data collected from a screen is processed locally (offline).</p>
