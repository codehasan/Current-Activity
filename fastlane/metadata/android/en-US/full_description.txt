<p><i>Current Activity</i> is an useful open source tool for Android Developers, which shows the package name and class name of current activity.</p><p>We use accessibility service api and package usage stats to monitor app activity changes and show the info in a freely moveable popup window.</p><p>How can this app help you? Well here are the main features of this app:</p><ul><li>It provides a freely moveable popup window to view current activity info</li><li>It supports text copying from popup window</li><li>It supports quick settings and app shortcut for easy access to the popup window. Meaning you can get the popup window in your screen from anywhere.</li></ul><p>Feeling uncomfortable giving all permissions?</p><li>You can disable accessibility permission and still use the app without any noticeable issue</li><li>But if you enable battery optimization or restrictions for this app, you might face some issues in runtime</li></ul>
